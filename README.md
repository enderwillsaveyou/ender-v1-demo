# Ender V1 Demo

A self-contained, data-free demonstration of Ender, a personal AI Chief of Staff interface.

## Try The Demo

Open the GitHub Pages site and start by messaging Ender. The five tabs at the bottom are:

- **Chat** — the default interface for asking Ender for information, opening a relevant view, or preparing a simulated task.
- **Dashboard** — startup review, system health, and one recommended next action.
- **Tasks** — prepare, approve, run, cancel, and retry fictional Codex or Claude tasks.
- **Files** — explore a mock AI OS folder structure with fictional file contents.
- **Test** — complete the seven-feature walkthrough and run the built-in privacy self-check.

## Safety Boundary

This repository contains static HTML only. It has no backend, API, analytics, external assets, account connection, real agent execution, machine control, or personal records.

Every status, task, reminder, approval, file, device, and health event is fictional. Interactive state remains in the browser and can be cleared with **Reset mock data**.

Do not enter real personal, health, account, or project information into the demo.
